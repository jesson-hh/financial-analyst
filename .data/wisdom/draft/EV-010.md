## 经验
A 股科技抱团牛市并非全员普涨, 而是资金沿产业链**接力高低切**, 一个细分炒高 → 切下一个未炒高的细分. 识别当前切到哪一棒, 能预判下一棒.

**2026 这一轮的接力脉络** (UP2 复盘):
```
光 (光模块/CPO) → 存储 (海力士/三星/美光带动) → 半导体材料 → 封测 (Tau 定律带动) → PCB (Rubin/CSC 架构高潮) → ...
```
- 规律: "抓住一个有利好催化的方向 → 极致抱团 → 炒高 → 切下一个低位细分"
- 不是炒一个时其他不动, 而是**当前棒猛涨, 其他温和**

## 适用条件
判断"我手里的细分还有没有戏" + "下一棒可能是谁". 板块轮动选股.

## 操作建议
- 手里细分**已经被炒过** (涨幅居前 + 资金流出) → 性价比下降, 考虑切换
- 关注"新信号方向 / 新消息催化"出现 → 该细分核心票通常有**几天涨幅**, 可早上车
- 不要在某细分炒到高潮 (如 PCB/Rubin 顶点) 时追入

## 反例 / 边界
- 高低切的前提是**整体抱团资金还在**(EV-002 集中度高); 一旦总资金撤离, 高低切链条断裂, 全跌
- 接力顺序事后看清晰, 事前预判下一棒有不确定性, 需结合催化事件 (如新架构发布)
- 与项目 `chain_kb/` 产业链知识库可结合: 用产业链上下游关系预判切换方向
