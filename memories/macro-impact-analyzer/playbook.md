# macro-impact-analyzer — 海外传导操作 playbook

## 4 种典型场景 + 操作

### 1. 美股强 + A 股弱 (背离 1)
**信号**: 美股 +1% 但 A 股 ≈ 0
**判读**: 内生韧性弱 / 北向慢半拍 / 板块切换中
**操作**: 防御板块抗跌, 等明日 follow-through 确认; 不追高

### 2. 美股弱 + A 股强 (背离 2)
**信号**: 美股 -0.8% 但 A 股 +0.5%
**判读**: A 股内生主线在跑 / 资金面好
**操作**: 主线主题加仓, 但警惕海外补跌带动情绪转弱 (尤其美科技股大跌)

### 3. 风险共振 (双跌)
**信号**: 美股 -1.5% + 港股 -1% + VIX > 25
**判读**: 全球避险, A 股大概率低开 1% 内
**操作**: 减仓 / 持仓改防御 / 避免追跌 / 黄金股 + 红利股 + 电力 中线机会

### 4. 流动性宽松 (双涨)
**信号**: 美股 +1% + 港股 +1% + VIX < 18
**判读**: 全球 risk_on, A 股大概率高开追随
**操作**: 主线加仓 (TMT 链 / 顺周期), 中小盘活跃度上升

## actionable_signals 写法

- **明日大盘**: "+/-X% 高/低开" + 置信度
- **重点板块**: "板块名 + 方向 + 1 句逻辑"
- **避免/关注 codes**: 只 cite memory 经验里反复出现的 ticker
- **决策时间**: "今晚收盘后" / "明日开盘 9:25-9:30" / "盘中 10:30"
