# ETF 评级体系 (五维评分)

> 核心原则: ETF 不是个股 — 消除单股阿尔法, 核心判断转向: 赛道方向对不对 + 资金在不在 + 价格是否合理

## AUM-tier 分层 (类比个股市值分层)

ETF 的 AUM 决定了持仓构成维度的可信度上限, 原理与个股市值分层一致:

| AUM 分层 | 持仓构成评分上限 | 理由 |
|---------|:----------:|------|
| 巨型 (>500亿) | 0 (不计分) | 成分股极度分散, 个股 alpha 被稀释为 beta, 无自下而上选股边际 |
| 大型 (100-500亿) | ±1 (打折) | 少量集中, 权重股质量有参考价值但影响有限 |
| 中型 (20-100亿) | ±2 (满分) | 成分集中度足够, 持仓质量评估有实质意义 |
| 小型 (<20亿) | ±2 (满分, 但需注意清盘风险) | 高集中度, 持仓有实质影响; 但 AUM 接近清盘线时 risk 维度需扣分 |

**巨型 ETF 的评级驱动力转向**: 持仓构成维度归零后, 评级主要由技术面 + 资金流 + 宏观/赛道判断驱动. 类似大盘股"因子面归零后靠技术面+消息面"的逻辑.

## 五维评分规则

每维度 -2 到 +2, 风控维度 -2 到 0; 总分 -10 到 +10.

### 1. 持仓构成 (holdings_score)

受 AUM-tier 限制 (见上表). 在限制范围内:

- 前十大成分股无重大负面事件 + 行业处于上升周期: +2 (小型/中型) 或 +1 (大型) 或 0 (巨型)
- 成分股质量一般, 行业中性: 0
- HHI > 0.30 (过度集中) + 头部持仓估值偏高: -1
- 单只成分股权重 > 25% + 该股近期负面: -2

### 2. 技术面 (technical_score)

与个股相同, 但 ETF 波动率更低, 信号更可靠:

- 价格在 MA20 以上 + MACD 金叉 + RSI 40-70: +1 ~ +2
- 价格在均线之间震荡: 0
- 价格跌破 MA60 + MACD 死叉: -1
- RSI < 30 超卖 (反弹机会): +1
- RSI > 70 超买: -1

### 3. 资金流 (flow_score)

ETF 特有维度 — 申赎数据直接反映机构/大资金的方向判断:

- 连续 10 日以上净申购 + AUM 持续增长: +2
- 近 5 日净申购, AUM 稳定: +1
- 申赎中性, AUM 横盘: 0
- 连续赎回 + AUM 持续萎缩: -1
- 近 30 日 AUM 缩水 > 20% + 连续赎回: -2 (清盘风险信号)

### 4. 估值/跟踪 (valuation_score)

- 折价 > 1% (价格 < NAV) + 历史分位低: +1 ~ +2
- 折溢价接近 0, 历史均值附近: 0
- 溢价 > 1% (价格 > NAV): -1 (均值回归压力)
- 溢价 > 2% 或跟踪误差 > 3%: -2
- 费率 > 1% 年化: 本维度最高 +1 (费率拖累抵消部分折价优势)

### 5. 风控 (risk_score) — 只能 0 或负值

- 无任何否决项 + 流动性充足: 0
- ADV/AUM < 1% (流动性偏低): -1
- 跟踪误差 > 3% 或 AUM < 5000万 (接近清盘): -1 ~ -2
- 杠杆/反向 ETF: 强制 -2 (长持结构性损耗)
- 任何硬否决触发: -2

## 总分评级

| 总分 | 操作建议 | 最大仓位 |
|------|---------|--------|
| ≥6 | 买入 | 10% |
| 4~5 | 逐步建仓 | 6% |
| 2~3 | 轻仓观察 | 3% |
| 0~1 | 观望 | 0-1% |
| -2~-1 | 不建仓, 已持有减仓 | 0% |
| ≤-3 | 回避/清仓 | 0% |

**veto_flag 触发 → 仓位强制 0%, 不论总分**

---

## 共振修正

- ≥ 4 个维度同向 (+或-): 额外 ±1 分
- 持仓面和技术面完全反向 (但 AUM 巨型时持仓面=0, 不触发矛盾折扣)
- 极端信号单维不独断: 任何单维 ±2 需至少 1 个其他维度同向支持
