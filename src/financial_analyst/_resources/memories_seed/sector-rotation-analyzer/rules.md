# sector-rotation-analyzer — 板块轮动判读

## 行业分类源

`stock_data/parquet/tushare_stock_basic.parquet` 的 `industry` 字段 (Tushare 一级行业, 共 60+ 类).

## 聚合阈值

- 同行业至少 **2 只**异动股才纳入排名 (否则单股噪声)
- 计算每行业的 avg_pct_chg 排序, top-5 = leaders, bottom-5 = laggards

## 典型轮动模式

| 模式 | leaders 特征 | laggards 特征 | 解读 |
|---|---|---|---|
| **顺周期 → 防御** | 银行/保险/煤炭/钢铁 | TMT/医药/消费 | 经济预期改善 + 利率上行 |
| **防御 → 成长** | 半导体/电子/计算机 | 银行/煤炭 | 风险偏好回升 |
| **普涨 + 红利领头** | 全行业 +1%↑ + 红利股领涨 | 几乎无 | 系统性反弹, 资金进场 |
| **普跌 + 防御股抗跌** | 红利/电力/食品 (相对) | 全行业 -1% | 系统性回调, 避险 |
| **AI 链分化** | 算力/芯片 | 软件/应用 | 主线退潮迹象, 注意切换 |

## 经验

- 同一日 leader/laggard 反向 — 风格切换信号, 第 2 日大概率延续
- 单板块连续 3 日 leader — 主线确认, 但要警惕游资接力末期
- 沪深 300 涨跌幅 < 0.3% 但板块分化大 — 结构性行情, 选股大于择时

## 输出约束

- rotation_signal 不超过 80 字
- 优先描述"今日 leaders / laggards 是谁", 其次"大盘背景"
- 行业名用 Tushare 标准命名 (银行/保险/证券/...)
