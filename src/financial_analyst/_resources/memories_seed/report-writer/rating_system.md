# 量化选股评级体系 v4

> 核心原则: 多维共振定方向, 单维极端降置信, 因子是概率不是真理, **大盘股因子要打折**

## 市值分层（v4 核心改进）

因子 IC 随市值衰减严重（2015-2026 全市场验证）:

| 因子 | 小盘(<100亿) | 中小盘(100-300亿) | 中盘(300-1000亿) | 大盘(>1000亿) |
|------|:----------:|:--------------:|:-------------:|:-----------:|
| amt_cv | **-1.06** | -0.53 | -0.29 | -0.15 |
| rev_20 | +0.59 | +0.37 | +0.27 | +0.09 |
| vol_20 | -0.67 | -0.45 | -0.31 | -0.07 |
| turnover | -0.54 | -0.30 | -0.12 | -0.02 |
| corr_cv_20 | -0.87 | -0.52 | -0.33 | -0.24 |
| big_up_freq | -0.81 | -0.51 | -0.35 | -0.12 |

**因子面评分必须先查市值分层，再决定权重：**

| 市值分层 | 因子面评分上限 | 理由 |
|---------|:----------:|------|
| 小盘 (<100亿) | ±2 (满分) | ICIR 0.5-1.0，因子有真实预测力 |
| 中小盘 (100-300亿) | ±2 (满分) | ICIR 0.3-0.5，仍有效 |
| 中盘 (300-1000亿) | ±1 (打折) | ICIR 0.2-0.3，预测力减半 |
| 大盘 (>1000亿) | **0 (不计分)** | ICIR <0.15，几乎无预测力 |

## 五维评分体系

每个维度 -2 到 +2 分, 总分 -10 到 +10。

### 1. 因子面（受市值分层限制）

**有效因子（23因子中经 IC 验证）:**
- amt_cv (ICIR=-0.80) — 成交额稳定性，低好（最强因子）
- corr_cv_20 (ICIR=-0.65) — 价量相关性，低好
- big_up_freq (ICIR=-0.63) — 大阳线频率，低好
- gap_dn_freq (ICIR=-0.62) — 跳空低开频率，低好
- obv_slope (ICIR=-0.55) — OBV斜率，负好
- rev_20 (ICIR=+0.52) — 反转，值大好
- big_dn_minus_up (ICIR=+0.51) — 大阴多于大阳，值大好
- turnover_20 (ICIR=-0.47) — 换手率，低好
- vol_20 (ICIR=-0.37) — 波动率，低好

**无效因子（仅参考不计分）:**
- MFI (IC=0.002)、筹码位置 (IC=0)、下影线 (IC=0.011)
- body_ratio (IC=0)、close_position (IC=-0.08)

**评分规则（受市值上限限制）:**
- 有效因子利多占比 ≥75%: +2（但大盘股→0，中盘→+1）
- 有效因子利多 > 利空: +1（大盘股→0）
- 均衡: 0
- 有效因子利空 > 利多: -1（大盘股→0）
- 有效因子利空占比 ≥75%: -2（大盘股→0，中盘→-1）

### 2. 技术面

- RSI < 30 超卖: +1 | RSI > 70 超买: -1 | 其他: 0
- MACD 金叉: +0.5 | 死叉: -0.5
- 多头排列: +0.5 | 空头排列: -0.5 | 纠缠: 0
- 四舍五入取整

### 3. 模型面（LightGBM 综合排名）

注意: 模型训练样本以小盘为主, 对大盘股的排名可靠性降低。

- 前 20%: +2
- 前 20-40%: +1
- 前 40-60%: 0
- 前 60-80%: -1
- 后 20%: -2

大盘股(>1000亿)模型面评分上限 ±1（因为模型本身对大盘预测力弱）。

### 4. 量能/主力面

- VR > 2.0 (强势控盘): +1
- VR > 1.5 (偏强): +0.5
- OBV 与价格同向 (共振): +0.5
- OBV 底背离 (价跌量涨): +0.5
- OBV 顶背离 (价涨量跌): -0.5
- VR < 0.5 (恐慌): -1
- 四舍五入取整, 上限 ±2

注意: 低换手公用事业股（核电/银行/电力）的 VR 绝对值天然偏高, 不应与成长股等同看待。VR>2.0 对核电股可能只意味着某天交易量偏大, 不等于"主力控盘"。

### 5. 消息/宏观面

- 明确事件催化 (政策利好/行业爆发): +1 ~ +2
- 板块当日涨幅 > 3%: +1
- 无显著消息: 0
- 板块当日跌幅 > 3%: -1
- 明确利空 (暴雷/制裁/行业下行): -1 ~ -2

注意: 催化逻辑链条超过 2 步的间接利好（如"中东→油价→能源替代→核电"）最多给 +1, 不给 +2。

## 总分评级

| 总分 | 评级 | 操作建议 |
|------|------|---------|
| ≥6 | ★★★★★ 强烈看多 | 重仓 (25%+) |
| 4~5 | ★★★★ 偏多 | 标准仓位 (15-20%) |
| 2~3 | ★★★☆ 中性偏多 | 轻仓试探 (10%) |
| 0~1 | ★★★ 中性 | 观望或极轻仓 (≤5%) |
| -2~-1 | ★★☆ 中性偏空 | 不建仓, 持有者减仓 |
| ≤-3 | ★★ 偏空 | 回避, 持有者清仓 |

---

## 共振修正规则

1. **共振加成**: ≥4 个维度同向 (+或-) → 额外 ±1 分
2. **矛盾折扣**: 因子面和技术面方向完全相反 → 总分向 0 收敛 1 分
   - 但大盘股因子面已为 0, 不触发矛盾折扣
   - 仅当因子面 ≥±1 且技术面方向相反时才折扣
3. **极端信号不独断**: 单一维度 ±2 不能独决, 至少 1 个其他维度同向支持

## V3 just_switched 金信号护盾 ⭐ (v4.1, 2026-05-07)

> **来源**: 立昂微 (SH605358) 2026-05-06 案例 — V3 半导体 just_switched=True 但本股因子 2:23 / RSI 77.9 / chip 89% / VR 0.0 多重周级滞后视图利空, 评 ★★★ 减仓. 5/7 实际 +4.89% 池中第一. 详见 [pitfalls.md "V3 just_switched 金信号 vs V4 仕佳风险冲突时 V3 优先" (2026-05-07)](pitfalls.md).

### 触发条件

`data.knowledge_pack.mainlines_today.just_become_mainline` 包含本股所属行业 (或主题归并后的等价行业):
- 元器件 / 半导体 / 通信设备 → 主题 **AI 硬件**
- 小金属 / 化工原料 / 电气设备 → 主题 **锂电**
- 行业内任一 just_switched=True → 主题归并组所有股都护盾激活

### 护盾规则

| 规则 | 内容 |
|------|------|
| **强制下限** | 评级 **≥ ★★★★** (即使五维总分 < 4 也不能低于 ★★★★) |
| **持仓建议禁忌** | 不允许 "减仓兑现 / 清仓回避" 等清仓向操作; 必须 "卫星 / 试探" 之一 |
| **操作 V10c** | 必须分批 (不允许 "全部止盈" 或 "全部清仓"), 留至少 1/3 仓让 V3 兑现 (主升 1-2 周) |
| **持有期最少** | 1 个月以上 (V3 月级 alpha 期典型 1-3 月) |

### 数据依据

- **R27 主线雷达 v1 12 月实证**: just_become_mainline 状态 fwd_60d **+5.54pp 胜率 87%** (n 较小但效应显著, p < 0.001 等价)
- **类似 mainline 持续状态**: fwd_60d +4.05pp 胜率 68% (n=364, t=9.08, p<0.0001)
- 月级前瞻 alpha 强于 5-20 日滞后视图 (因子面 / RSI / VR / chip 等)

### 例外情形 (放宽护盾)

仅在以下三种**铁底信号**触发时, 评级可下调至 ★★★, 但必须在结论中**显式说明"V3 金信号护盾被技术面铁底信号触发例外"**:

1. RSI > 90 (极端超买, 历史回撤概率 > 80%)
2. chip 区间位置 > 95% (筹码全部锁定在顶部, 抛压释放在即)
3. 单日 -10% 暴跌或连续 3 日累计 -15% (V4 信维候选反指, 主升期破位)

### 与其他规则的优先级

护盾规则**仅在 V3 just_switched 命中时激活**, 不影响以下规则:
- 大盘股 (>1000亿) 因子面归零 — 仍生效
- 中盘股 (300-1000亿) 因子面打折 — 仍生效
- 共振加成 / 矛盾折扣 — 仍生效, 但最终评级不能低于护盾下限 ★★★★

### V3 mainline 持续中 (just_switched=False) 的退化规则

V3 已老 (mainline 状态持续 > 1 个月), 月级 alpha 接近兑现 → V4 仕佳风险可与 V3 平等权衡, 评级可下调至 ★★★. 此时 V4 警示生效, 不享受金信号护盾.

## 市值分层的实际影响示例

### 大盘股 (>1000亿): 比亚迪/平安/宁德/核电

因子面强制为 0, 模型面上限 ±1 → 评级主要由技术面+量能面+消息面驱动。
最高可能得分: 0+2+1+2+2 = +7 (但实际很少所有维度都+2)
这意味着大盘蓝筹很难获得 ★★★★★, 通常在 ★★★~★★★★ 区间, 符合其低 beta 属性。

### 小盘股 (<100亿): 奥瑞德等

因子面满分 ±2, 因子信号权重最高。因子多空比是核心驱动力。
但需注意: 小盘投机股的涨停/跌停频率高, 单日极端波动不否定因子统计规律。

### 中盘股 (300-1000亿): 立讯精密等

因子面上限 ±1, 因子有参考价值但不决定方向。
综合性最好 — 因子有效但不过度依赖, 技术面和模型面权重平衡。

## V4 仕佳模式 4 要素硬规则 ⭐⭐ (v4.2, 2026-05-13)

> **来源**: SZ002498 汉缆股份 5/8 → 5/13 误判案例. 5/8 我用 ret_60d +77% 单一维度判 V4 仕佳, 评 ★★★ 观察. 5/12 + 5/13 连续 2 涨停, 累计 **+20.7%**. 详见 [pitfalls.md "V4 仕佳模式 4 要素 + V8 极端信号读取 + V3 板块切换感知" (2026-05-13)](pitfalls.md).

### V4 仕佳模式定义 (修正后)

V4 仕佳模式的**完整 4 要素**, 必须**同时满足**才能判定:

| 要素 | 内容 | 量化判据 |
|------|------|----------|
| **(a) 涨幅** | ret_60d 已涨 6 月+ | ret_60d ≥ +30% (单维度) |
| **(b) 催化兑现完** | 主要利好已落地, 无新催化 | 一次性事件型 (年报披露后/政策出台后/重大订单签订后); **业绩持续兑现的不算** (国网订单陆续/客户 IPO 推进等) |
| **(c) V5 反应弱** | 利好高开低走 / 利空低开低走 | 近 3-5 日内有具体反应模式案例; 高开高走涨停**不算** V5 反应弱 |
| **(d) V3 板块退潮** | 行业从 mainline 退到 cold/decay | data.knowledge_pack.mainlines_today 中本股行业 status ∈ {'cold', 'decay'}; mainline 持续中**不算**退潮 |

### 4 要素硬校验规则

| 满足要素数 | 判定 | 评级影响 |
|------------|------|----------|
| 4/4 全部满足 | **真 V4 仕佳模式** | Bear 论点成立, 评级可下调到 ★★, 操作"减仓兑现 / 清仓回避" |
| 3/4 (缺 1) | **仕佳临界期** | Bear 论点权重折扣 50%, 评级可下调 1 档但不可一步到 ★★ |
| 2/4 (缺 2) | **正常调整或主升中段** | Bear 论点 V4 仕佳判定**失效**, 不影响评级 |
| 0-1/4 | **不是仕佳** | Bear 论点 V4 仕佳判定**完全失效**, 必须改判为"主升加速期"或"正常上涨" |

### V8 极端信号配合判读 (强制)

V4 仕佳 4 要素判定时, 必须同时检查 V8 极端信号方向:

| V8 信号 | 方向判读 | 处理 |
|---------|----------|------|
| **VR > 5 + OBV 上升 + MFI > 70 + 主力吸筹** | **真控盘看多** | V4 仕佳判定**直接否决** (即使 4 要素满足也不能减仓) |
| VR < 0.3 + chip > 90% + 主力 5d 累计净流出 > 5 亿 + 龙虎榜机构净卖 | 真派发看空 | V4 仕佳判定**强化** (即使 4 要素只满足 3 也可减仓) |
| **VR > 5 或 VR < 0.3 (单维异常)** | 不允许标"数据噪声"忽略 | 必须配合其他维度判读真实方向 |
| chip > 95% 或单日 -10% 或 RSI > 90 | 铁底信号 | 与 V3 just_switched 护盾的例外规则一致 |

### 实战案例

**汉缆股份 5/8 (反例)**:
- (a) ret_60d +77.78% ✓
- (b) 国网订单**持续兑现** (2 月 1.08 亿, 5 月预中标 2.44 亿) ✗
- (c) 5/12 高开 +5% **高走涨停** ✗
- (d) V3 电气设备 mainline **加速** +12.91pp (5/12) ✗
- **结论**: 1/4 满足, **不是仕佳**, 是主升加速期. 5/12 + 5/13 连 2 涨停 +20.7% 印证.
- 同时 VR=37.6 + OBV 上升 + 主力吸筹 = **真控盘看多**, V4 判定被直接否决.

**SH600666 奥瑞德 5/8 (正例)**:
- (a) ret_60d +63.71% ✓
- (b) 算力业务故事已发酵 4 周, 无新催化 ✓
- (c) 5/6 涨停后 5/7 未连板 + 5/12 大盘大涨日 -5.6% ✓
- (d) 元器件 lu_max 727 亿大龙警戒 (虽 mainline 但接近顶部) ⚠️ 3.5/4
- **结论**: 4 要素近乎全满足, 真仕佳模式. 加 V8 派发 (主力 4 周 -10.6 亿) → 评 ★★ 清仓回避正确, 5/8 → 5/12 累计 -5.6% 印证.

### 与其他规则的优先级

V4 仕佳 4 要素硬规则的优先级:
- **低于** V3 just_switched 金信号护盾 (v4.1) — 金信号在时仕佳判定直接失效
- **高于** 单维因子/LGB 反向看空信号
- **同级** V8 极端信号判读 — 两者同时检查, V8 派发证据加重仕佳, V8 控盘证据否决仕佳

## 涨停日 5 重共振判据 ⭐⭐⭐ (v4.3, 2026-05-14)

> **来源**: SH600666 奥瑞德 5/13 涨停 + 5/14 炸板 vs SZ002498 汉缆股份 5/12 涨停 + 5/13 续板 同日对照案例. 同样"涨多+调整"的两只票, 次日反应完全相反, 表层因果不足以解释, 深度根因是**5 重共振**的方向. 详见 [pitfalls.md "涨停日强弱判读 5 重共振" (2026-05-14)](pitfalls.md).

### 5 重共振判据矩阵

涨停日 T+1 反应预判, 检查 5 个维度:

| 维度 | **连板信号 (正向)** | **一日游信号 (反向)** | 数据来源 |
|---|---|---|---|
| **① 筹码结构** | 60d 持续上涨无密集套牢区 | 60d 内顶部巨量换手 (单日 > 25% 连 3 天) → 上方套牢盘等解套 | qlib daily turnover_rate 滚动 |
| **② 主力面** | 5d 主力净流入 > 0 + **VR > 2 吸筹** + OBV 上升 | 5d 主力净流出 > 5 亿 + **VR < 0.5 派发** + 龙虎榜机构净卖 | TDX F10 lhb_seats / VR / OBV |
| **③ 催化面** | 持续新催化 (新订单 / 业绩预增 / 技术突破) | 老故事 4+ 周无新催化 (一次性事件已兑现) | TDX F10 recent_events / WebSearch |
| **④ V3 板块** | mainline 中段 + lu_max < 500 亿 (健康主升) | mainline 后段 + **lu_max > 700 亿大龙警戒** (R27 反指 -1.5pp) 或 V3 退潮 | mainlines_today.json |
| **⑤ V5 涨停质量** | gap > 3% 高开 / 秒板 (10:00 前) / 封单 > 流通市值 3% / seal_at_close=True | gap < 1% 平开 / 尾盘封板 (seal_bar > 24) / 弱封 / seal_at_close=False | board_seal_micro (R12 微结构) |

### 判定规则

| 正向维度数 | 判定 | T+1 操作建议 |
|------------|------|--------------|
| **5/5 全正向** | 强烈连板 (汉缆 5/12 案例) | 持仓不动, 可加 1/3 仓 (但不超 1/3) |
| 4/5 正向 | 连板高概率 | 持仓不动观察 |
| 3/5 正向 | 普通涨停 (50/50) | 持仓守支撑, 等 T+1 反应模式确认 |
| **3+/5 反向** | **一日游高概率** | T+1 早盘冲高即兑现, 未持有不入 |
| **5/5 全反向** | **必然炸板 (奥瑞德 5/13 案例)** | T+1 必清仓, 早盘任何反弹兑现 |

### 派发期内涨停的特殊处理

当 ② 主力 5d 累计净流出 > 5 亿 + VR < 0.5 (V8 派发证据完整) 时, 涨停大概率是**出货板** (主力借涨停清仓):
- 即使 ⑤ V5 形态强 (gap > 3% / 秒板) 也**不构成连板信号**
- T+1 大概率炸板, **不参与**
- 整体趋势判断不变, 单日反弹不构成清仓回避建议的撤回理由

### 与其他规则的关系

| 关系 | 说明 |
|------|------|
| **与 v4.1 V3 护盾** | V3 just_switched 时 ④ 板块维度自动 = 强正向, 5 重共振大概率全正 → 涨停连板 (立昂微 5/6) |
| **与 v4.2 V4 仕佳** | V4 仕佳 4 要素满足时 5 重共振大概率全反 → 涨停后炸板 (奥瑞德 5/13). 两者**互为微观/宏观表现** |
| **优先级** | 涨停日单独使用此判据; 评 V4 仕佳时**必须同时检查 5 维**; V3 金信号护盾**仍最高优先级** |

### 实战案例对照

**奥瑞德 5/13 (5/5 全反向 → 炸板)**:
- ① 筹码: 4/16-4/17 顶部 7.16 (3 天换手 80%+), 5/13 涨停仅 6.07 距顶 -15%, 上方大量套牢
- ② 主力: 4 周累计 -11.93 亿派发 + VR 0.1
- ③ 催化: 算力老故事 4 周无新催化 + PE 114 估值脱节
- ④ 板块: 元器件 mainline + lu_max 727 亿大龙警戒
- ⑤ V5: 弱涨停 (次日炸板印证)
- **5/14 实际**: -3.95% 炸板 ✓ 判据有效
