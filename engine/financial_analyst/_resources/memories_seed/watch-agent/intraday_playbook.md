# 盘中盯盘决策守则 (watch-agent 必守)

> 蒸馏自项目**已验证**知识 (R7-R20 情绪研究 / pitfalls / rating_system / V1-V10 视角)。
> 盘中单股 advisor 下结论前必须遵循；与触发原因冲突时，以本守则的风控/失效红线优先。

## 一、核心逻辑：反转 > 动量
- **反转是 A 股核心** (rev_20 是最强因子)。**不要追涨动量**，不因"涨得多 / 走势强"就看多。
- "已涨多 (ret_20d 高) + 放量" 是**见顶/派发**风险信号，不是强势确认。

## 二、量能 regime 警示 (验证过的减仓信号，遇到优先看空)
- **super_distr**：近 20d 涨幅≥10% 且 换手放量≥2.5x 且 尾盘 30min 拉升 → 月度 11/12 SS 级，后 5 日预期 **−4.2pp** → **强烈倾向 reduce/sell，绝不追**。
- **distr**：涨幅≥10% + 放量 → 后 5 日 −1.4pp，谨慎。
- **tail_surge**：尾盘急拉 + 远离均价 → 后 5 日 −1.4pp，视为**派发**不是突破。
- 反向偏多：缩量回调 (down_mild+爆量 / down_strong+缩量) 后 5 日 +0.9pp。

## 三、涨停 / 封板强弱 (board v5)
- **强**：秒板 / 一字板 (gap≥9%) / 10:00 前封板。
- **弱**：尾盘 (≥14:00) 封板 / **收盘破板** (monster 率仅 2.6%) / 当日多次破板。
- 首板仅在 total_score≥4 (四维 + 封板微结构) 才考虑参与。

## 四、游资博弈票 — 模型失效红线
- 特征：小盘 (<200亿) + PE>100 + ret_60d>50%。
- **量化模型对这类票结构性失效**：buy/add 信号**仅供参考**，不可重仓；研判必须明示"游资博弈票，模型信号仅供参考"。

## 五、市值分层 (因子可信度随市值衰减)
- 大盘 (>1000亿)：**因子面归零**，不要用因子看多大盘股。
- 中盘 (300-1000亿)：因子打折。
- 小盘 (<100亿)：因子面最可信。

## 六、风控铁律 (高于一切)
- **T+1**：今买明卖；**涨跌停不成交**。
- **负向事件** (立案 / 处罚 / 减持 / 业绩预减 / 退市风险) → 持仓优先 reduce/sell，空仓**禁建仓**。
- 触发 **stop_break** (跌破用户止损) → 严格 sell，不找理由扛单。
- 系统性下跌行情中，个股因子 / 技术信号普遍失效 → 整体降级。

## 七、V1-V10 视角锚 (精简，结论尽量挂 V# 锚点)
V1 主营是否真景气 · V2 业绩能否兑现 · V3 估值是否透支 · V4 困境反转(立讯模式) · V5 主力是否进场 · V6 题材催化真伪 · V7 筹码是否松动 · V8 大盘/主线环境 · V9 流动性/情绪 · V10 执行纪律(止损不犹豫)。
